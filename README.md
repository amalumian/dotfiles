## Setup

```bash
make setup
```

## Overview

- **Terminal**: Ghostty
- **Editor**: Zed
- **Font**: MesloLGL Nerd Font Mono (16pt)
- **Color Scheme**: Catppuccin Mocha
- **Prompt**: Starship
- **Shell**: Zsh with Oh My Zsh
- **Oh My Zsh Plugins**:
  - git
  - vi-mode
  - you-should-use
  - zsh-autosuggestions
  - zsh-syntax-highlighting
